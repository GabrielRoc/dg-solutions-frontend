## Descrição

Implementação de um front-end para uma lista de pessoas com nome, data de nascimento e idade(gerada)

## Instalação

```bash
# setting up environment variables file
$ cp .env.example .env

# installing dependencies
$ yarn
```

## Execução

```bash
# development
$ yarn dev

# production mode
$ yarn build
$ yarn preview
```
